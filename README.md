# Tanvi Host Front-End Web Page

Welcome to the front-end web page for *Tanvi Host*. This is a modern, responsive web project built with React + Vite.

---




## ⚙️ Tech Stack

- **React** (for UI/JS logic)
- **Vite** (for build tooling & dev server)
- **ESLint** (linting & code standards)
- HTML5, CSS3 (and optional Tailwind or custom global CSS)
- JavaScript (primary language)
